# AGENT-44 — Steam store page & press kit (fully fledged §G)

## Source
[`docs/FULLY_FLEDGED_EXPERIENCE.md`](../docs/FULLY_FLEDGED_EXPERIENCE.md) axis **G — Discovery**.

## Objective
Produce **repo- or drive-stored** assets + copy templates (even if Steam partner UI is external):

1. **Capsule** (231×87, 616×353, 460×215, etc. per [Steam guidelines](https://partner.steamgames.com/doc/store/assets))
2. **Library hero** / header if required
3. **≥5 screenshots** (capture at fixed resolution; no debug overlay)
4. **Short description** (≤300 chars rough) + **long description** (features, modes, controls)
5. **Tags** list (genre: action, arcade, roguelite, etc.)
6. **Press kit** folder: logo SVG/PNG, one-page fact sheet (`docs/PRESS_KIT.md` or `press/README.md`)

## Acceptance criteria
- [ ] `docs/STEAM_STORE_CHECKLIST.md` with **checkboxes** and **file paths** or “uploaded to partner” notes.
- [ ] Screenshots **readable** at store thumbnail size.

## Out of scope
Actual Steam partner clicks (human); pricing strategy.
